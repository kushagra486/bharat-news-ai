# 🇮🇳 Bharat News AI

**Live AI-powered daily news intelligence — Built for India, Powered by Groq**

![Bharat News AI](https://images.unsplash.com/photo-1524492412937-b28074a5d7da?w=1200&q=80)

## 🚀 Features

- 📡 **4 Live News Sources** — GNews, NewsData.io, The Guardian, NewsAPI.org
- 🤖 **Groq AI Analysis** — Llama 3.3 70B: summaries, sentiment, timeline, key points
- 🇮🇳 **India-first** — Dedicated India tab + Hindi news feed
- 📱 **Instagram-style UI** — Stories, tiled feed, dark/light mode
- 🔔 **Keyword Alerts** — Browser push notifications on keyword match
- 🔬 **Deep Research** — Ask any question, get AI-powered analysis
- 📖 **Article Reader** — Full in-app reader with TTS audio
- 📸 **Share as Image** — Export any article as a branded PNG
- 🗂️ **AI Story Clustering** — Groups related articles automatically
- 📲 **PWA Installable** — Add to Home Screen on mobile
- 🌙 **Dark/Light Mode** — Persisted across sessions
- ⚡ **100% Free** — Groq free tier + free news APIs

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Vanilla HTML/CSS/JS |

2. Open `index.html` in your browser — or deploy to Vercel
3. All API keys are pre-configured
